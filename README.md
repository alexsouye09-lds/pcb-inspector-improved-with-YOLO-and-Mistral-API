# pcb-inspector-improved-with-YOLO-and-Mistral-API
Description du projet  PCB Inspector est un pipeline intelligent de détection et d’analyse de défauts sur des cartes de circuits imprimés (PCB) pour un usage industriel. Le projet combine Computer Vision et IA conversationnelle afin de générer automatiquement un plan d’action précis pour réparer ou contrôler les cartes.
